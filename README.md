<h2>Note: Please Do Not Sync This Repo, Else It Will Corrupt!</h2>

<h2>Project Overview</h2>
Mango is an online food ordering system which have frontend in ASP.NET MVC. It uses 7 microservices to handle orders.

It stores data in MS SQL Serevr and uses RabbitMQ to provide event-driven architecture.

<h2>Setup Guide</h2>

1. Install .NET SDK
2. Pull & spin up SQL Server Container
3. Pull & spin up RabbitMQ Container
4. Clone Repo
5. dotnet restore
6. dotnet build
7. Press F5
